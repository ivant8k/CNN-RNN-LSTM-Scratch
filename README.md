# CNN-RNN-LSTM-Scratch

Repositori ini berisi implementasi algoritma *Convolutional Neural Network* (CNN), *Recurrent Neural Network* (RNN), dan *Long Short-Term Memory* (LSTM) yang dibangun dari awal (*from scratch*) untuk memahami cara kerja dasar dari setiap arsitektur.

## Cara Setup dan Menjalankan Program

### Persyaratan Sistem
Pastikan Anda telah menginstal **Python 3.x** di sistem Anda.

### Instalasi
1. *Clone* repositori ini ke komputer lokal Anda:
   ```bash
   git clone https://github.com/ivant8k/CNN-RNN-LSTM-Scratch.git
   cd CNN-RNN-LSTM-Scratch
   ```
2. (Opsional) Buat *virtual environment* dan aktifkan:
   ```bash
   python -m venv venv
   
   # Untuk Windows:
   venv\Scripts\activate
   
   # Untuk Linux/Mac:
   source venv/bin/activate
   ```
3. Instal semua dependensi yang dibutuhkan:
   ```bash
   pip install -r requirements.txt
   ```


### Menjalankan Program dengan UV
Menjalankan dengan UV

Jika menggunakan tool `uv`, berikut langkah dasar menjalankannya:

- Instal `uv` (opsional):

   ```bash
   pip install uv
   ```

- Sync dengan librari proyek:

   ```bash
   uv sync
   ```
